#DESCRIPTION

This task focuses on building a **Convolutional Neural Network (CNN)** for image classification using TensorFlow.

A CNN is a type of deep learning model specifically designed to process and analyze visual data such as images. It uses layers like **convolutional layers, pooling layers, and fully connected layers** to automatically learn important features from images.

In this project, the model is trained on an image dataset (such as CIFAR-10), where each image belongs to a specific category. The CNN learns patterns such as edges, shapes, and textures to correctly classify images into different classes.

The workflow includes:

* Loading and preprocessing the dataset
* Building the CNN architecture
* Training the model on training data
* Evaluating the model on a test dataset
